Project Overview
The Task & Team Project Management System is a full-stack web application that helps teams organize projects, assign tasks to members, and track task progress efficiently.
The platform provides a dashboard where users can manage projects, create and assign tasks, and monitor project progress using a Kanban-style board.
This project is built using HTML, CSS, JavaScript, Node.js, Express.js, and SQL.
Features
• User authentication system (Signup/Login)
• Create, update, and delete projects
• Create and assign tasks to team members
• Track task progress using Kanban board
• Dashboard showing project and task statistics
• Responsive and clean UI

<h2>Project Overview</h2>
The Task & Team Project Management System is a full-stack web application that helps teams organize projects, assign tasks to members, and track task progress efficiently.</br>
The platform provides a dashboard where users can manage projects, create and assign tasks, and monitor project progress using a Kanban-style board.</br>
This project is built using HTML, CSS, JavaScript, Node.js, Express.js, and SQL.</br>
<h2>Features</h2>
• User authentication system (Signup/Login) </br>
• Create, update, and delete projects </br>
• Create and assign tasks to team members </br>
• Track task progress using Kanban board </br>
• Dashboard showing project and task statistics </br>
• Responsive and clean UI
